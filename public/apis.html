API Reference

Complete documentation with 13 endpoints. Test directly in your browser.

Introduction

Get started with VanFact API

VanFact API adalah layanan REST API yang menyediakan 13 endpoint untuk kebutuhan aplikasi modern. Dibangun dengan performa tinggi, rate limiting, caching, dan mudah diintegrasikan.

Base URL
https://your-domain.vercel.app/api

Authentication

Secure your API requests

Semua endpoint memerlukan API Key untuk autentikasi. Sertakan API Key pada header setiap request. Rate limit: 10-60 requests per minute tergantung endpoint.
