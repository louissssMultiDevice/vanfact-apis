# VanFact API

Powerful REST API Service dengan berbagai endpoint untuk developers.

## Features

- 🌍 **Planet API** - Informasi planet dengan visualisasi gambar
- 🕌 **Pray Reminder** - Jadwal sholat harian
- 🤖 **DeepSeek AI** - AI text generation
- 📱 **Social Media Downloaders** - TikTok, Instagram, YouTube, Facebook, Twitter
- 🎵 **Spotify** - Track information
- 📁 **MediaFire** - Direct download links
- 🔍 **Google Search** - Web scraping
- 📊 **Status** - Realtime system monitoring
